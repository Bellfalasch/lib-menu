# lib-menu

## Usage

### Gradle build script

    repositories {
        maven {
            url 'http://repo.enonic.net/public'
        }
    }

    dependencies {
        include 'com.enonic.lib:menu:1.0.0'
    }

### Controllers

    var menu = require('/lib/enonic/menu/menu');
